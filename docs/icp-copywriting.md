## Framework copywriting: PAS + Proof

Pour chaque section produit, utiliser:

1. **Problem** — nommer la douleur en Darija
2. **Agitate** — amplifier l'émotion
3. **Solution** — présenter le produit comme LA réponse
4. **Proof** — avis, chiffres, certifications, science
5. **CTA** — action claire avec scarcity

### Exemple complet (Bague Lien Éternel):

```
[PROBLEM]
كتشري bijoux w mn ba3d chhar kayb9aw k7lin? 
Kat7essi blam m3a l'outfit dialk?

[AGITATE]
Chaque fois li kat7ebi ttsawri, kat7ayd yedik. 
Hadchi ma kaynch m3a GUMÜÇROYAL.

[SOLUTION]
Bague Lien Éternel — le maillon iconique réinventé.
Zircons taille brillant + pierre solitaire au centre.
Acier inoxydable doré à l'or fin.

[PROOF]
★★★★★ 4.9/5 — 847 avis
✓ Certifié acier 316L
✓ Testé 10,000 cycles sans ternissement
✓ +2,341 commandes ce mois

[CTA + SCARCITY]
Commande daba — khmsa khatiwat bqa f stock dial Casa
[أضيفي للسلة — 249 MAD]
```

## Règles copy pour l'AI Coder

1. Tout texte visible utilisateur = **Arabe RTL** (avec mots Darija/Fr intégrés)
2. `dir="rtl"` sur `<html>` et composants
3. Font arabe: **Tajawal** ou **Cairo** (Google Fonts)
4. Font logo/titres premium: **Playfair Display** ou **Cormorant Garamond**
5. Ne jamais utiliser "dropshipping", "AliExpress", "supplier"
6. Toujours "By GUMÜÇROYAL" après le nom produit
7. Prix en **MAD** avec format: `249 د.م.`
